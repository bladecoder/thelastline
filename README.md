# The Last Line
Interactive fiction horror game inspired by Tetris mechanics

## Technical details

- Follow most of NES Tetris rules
- Lock down time not implemented.
- ARE/entry delay allow move and rotate pieces as in NES.

## GAME TYPES

### Marathon

- 30 levels
